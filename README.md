#DevPortfolio — Site Web de Portfolio Personnel
#Contexte du projet

Le projet DevPortfolio a pour objectif de concevoir et développer un site web personnel permettant de présenter le parcours, les compétences et les réalisations du créateur.
Il se divise en deux grandes étapes : la phase de conception sur Figma et la phase de développement avec HTML et CSS.

Phase de conception

En tant que concepteur, l’objectif est d’analyser la maquette Figma de la page d’accueil et de respecter l’identité visuelle : couleurs, typographie et mise en page.

#Pages à concevoir

Accueil : photo de profil, nom et titre professionnel

À propos : présentation personnelle et compétences

Projets : mise en avant des réalisations avec descriptions, technologies utilisées et liens

Contact : formulaire interactif pour contacter le créateur du site

En tant que développeur, l’objectif est de transformer la maquette Figma en un site web responsive et fonctionnel en utilisant HTML et CSS.

Tâches principales

Intégrer l’ensemble des pages : Accueil, À propos, Projets, Contact et éventuellement Formations & Certifications

Ajouter des animations au défilement et une barre de navigation fixe pour améliorer l’expérience utilisateur

Vérifier la conformité du code HTML avec le validateur W3C

Structurer le code de manière claire, lisible et réutilisable afin de faciliter la maintenance et la personnalisation

#Contenu du site
Accueil

Photo de profil

Nom et titre professionnel (exemple : Développeur Front-End)

Liens vers l’email, LinkedIn, GitHub et le CV (format PDF)

À propos

Courte présentation personnelle

Compétences avec barres de progression animées

Technologies maîtrisées présentées sous forme de logos ou de texte

Projets

Présentation des projets sous forme de cartes

Description, objectifs et technologies utilisées

Liens vers la démo en ligne et le code source sur GitHub

Formations & Certifications (optionnel)

Présentation sous forme de frise chronologique verticale

Contact

Informations : email, téléphone, localisation

Formulaire fonctionnel avec champs nom, email, message

Fonctionnalités principales

Design moderne et professionnel

Site entièrement responsive (mobile, tablette, ordinateur)

Animations fluides au défilement

Barre de navigation fixe sur toutes les pages

Code clair, bien structuré et facile à personnaliser

Accessibilité optimisée pour la recherche de stage ou d’emploi

Prototype sur Figma
